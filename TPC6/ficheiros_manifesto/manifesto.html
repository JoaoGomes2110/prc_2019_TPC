Resumo:

Neste trabalho para casa foi pedido que juntassemos a ontologia da aula o conceito de Freguesia.

Foi utilizado o protegé para modelar, e depois um stylesheet xslt para transformar o xml em turtle.

O ficheiro dado na aula em xml já incluia as freguesias o que suscitou alguma duvida, no entanto utilizei a stylesheet nesse ficheiro.

O ficheiro dado pelos dados.gov.pt passei do formato xslx para xml com auxilio de um script em node para correr basta, instalar as dependências

Estrutura da pasta de Entrega:

-Stylesheet e Ficheiro Turtle gerado pela mesma na pasta -> Stylesheet_e_Resultado;

-Ficheiro Criado a partir do Protegé na pasta -> FicheirosProtegé;

-Junção dos dois ficheiros a partir da linha de comando, sendo este o resultado final na pasta -> ResultadoCat;

- Script node na pasta inicial chamado xslx2xml.js
